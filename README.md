# Digital Clock

A beautiful and responsive Digital Clock built with HTML, CSS, and JavaScript.

## Features

* Real-time digital clock
* Current date display
* Modern Glassmorphism UI
* Animated background
* Responsive design for mobile and desktop
* Lightweight and easy to customize

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Preview

The clock displays:

* Current hours, minutes, and seconds
* Current date
* Modern glass-style card with blur effect

## How to Run

1. Download or clone the repository:

```bash
git clone https://github.com/PhanathEM/DigitalClock-Project.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in your browser.
